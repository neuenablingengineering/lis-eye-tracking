findEyeCenter(gray_eye, thresh_eye)¶

Approximates the center of the pupil by selecting the darkest point in gray_eye that is within the blob detected in thresh_eye

Parameters:
  • gray_eye – Gray-scale image of eye
  • thresh_eye – binary eye image with contours filled in
Returns:

The pixel location of pupil center
Return type:

tuple(int,int)
getPupilAvgFromFace(gray_face, eyes, x, y, w, h)¶

Takes a grey version of face image, a list of eyes, and the x, y, w, h, coordinates of the face. Returns a list [x, y] for the average between the eyes contained in the list.

Parameters:
  • gray_face – A gray-scale version of the face image
  • eyes – A list of eyes located within image
  • x – The x coordinate of the face
  • y – The y coordinate of the face
  • w – The width of the face
  • h – The height of the face
Returns:

The average center position of the eyes in the list
Return type:

tuple(int,int)
